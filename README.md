# Compression d'image via quadtrees

## Installation

Pour installer le projet, extraire l’archive, entrer dans le dossier extrait et lancer la commande :
```$> make```
Les commandes make clean, make fclean et make re sont disponibles. (respectivement : supprime les fichiers objets, supprime les fichiers objets et l’executable et supprime les fichiers objets et l’executable et recompile le projet.) Ajoutez les images que vous voulez essayer dans le dossier img puis lancez la commande :
```$> ./quadtree```

## Utilisation

Une fois l’application lancée vous aurez une petite sélection d’images sous les yeux, cliquer sur celle sur laquelle vous voulez éffectuer vos tests. L’image se charge sans perte, vous avez ainsi accès à 6 boutons différents :
- Save as bin in color : Sauvegarde du quadtree en binaire et en couleur dans un fichier .qtc dans le dossier img
- Save as bin in b & w : Sauvegarde du quadtree en binaire en noir et blanc dans un fichier .qtn dans le dossier img
- Save as graph in color : Sauvegarde du quadtree en graph en couleur dans un fichier .gmc dans le dossier img
- Save as graph in b & w : Sauvegarde du quadtree en graph en noir et blanc dans un fichier .gmn dans le dossier img
- Minimisation : Minimise l’arbre avec perte.
- Open img : Ouvre une image ou un fichier encodé

◆ quadtree_maker2()

void quadtree_maker2 ( t_lc **  l,
MLV_Image *  img,
t_qt **  qt,
int  operations 
)

Create a quadtree based on img. Calculates the distance between the average color of the actual zone and each of the pixels composing the image at that specific zone. Keeps every distance in a sorted double linked list contained in a list_container l. The first node of the list is the one we have to divide in 4.

Parameters
llist container with a pointer on the first element of a list used to get the worst distance.
imgthe image used to create the quadtree.
qtthe quadtree created.
operationsthe number of operations (the lower, the lesser the quality of the image)
